**AutoRecon** is a lightweight Bash script designed for beginner to intermediate Linux users to perform a basic system audit. It gathers key system information, checks user activity, analyzes network settings, and scans for outdated packages â€” all from one script.

---

## Features

- System information overview (hostname, OS, kernel, uptime, CPU, memory)
- Logged-in users and all system users
- Network details and open ports
- Basic firewall status (UFW)
- Outdated package detection (APT)
